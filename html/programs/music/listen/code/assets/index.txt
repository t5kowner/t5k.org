Title=MIDI Information

This directory contains my routines to make MIDI files from
<UL>
  <li><a href="midi.cgi">character strings</a> (such as "Have a nice day.")
  <li><a href="primes.cgi">prime numbers</a> (see the <a href="/programs/music/listen/">listening guide</a>)
  <li><a href=sequence.cgi>integer sequences</a> (such as {1,2,3,4,5,6,...} -- unfinished!)
</UL>

If you are interested in the technical aspect of the midi standard, see the
following references from <a href="http://sunsite.unc.edu/emusic-l">EMUSIC-L</a>:
